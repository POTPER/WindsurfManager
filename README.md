# Windsurf Account Manager

Chrome 扩展，用于管理和切换多个 Windsurf 账号会话，自动跟踪配额用量。

## 功能

- **会话捕获** — 保存当前 Windsurf 账号的 cookies 和 localStorage
- **一键切换** — 恢复已保存的会话，自动刷新用量数据
- **智能换号** — 评分制自动选择最优账号（`free` 不参与；优先快过期 trial、轮换使用、综合日/周配额）
- **自动刷新** — 每 10 分钟并发刷新所有账号用量，倒计时显示下次刷新时间
- **用量展示** — 日配额、周配额、剩余天数环形图，分组显示（Pro / Trial / Free）
- **Token 失效检测** — 自动标记 token 过期的账号，卡片半透明显示
- **导出/导入** — JSON 格式备份和恢复会话数据，导入时自动去重

## 下载

- **下载发布版**
  1. 打开本仓库的 [Releases](../../releases) 页面
  2. 下载最新版本的 `.zip` 文件
  3. 解压到任意目录（路径不要带中文/空格）

## 加载扩展

以 Chrome 为例（Edge / Brave / Arc 等 Chromium 浏览器步骤一致）：

1. 打开浏览器，地址栏输入 `chrome://extensions` 回车
2. 打开右上角的「开发者模式」开关
3. 点击左上角「加载已解压的扩展程序」
4. 选择刚才解压或克隆得到的项目文件夹
5. 加载成功后，建议点击工具栏的拼图图标 → 把扩展「📌 固定」到工具栏

## 使用

- **添加账号** — 登录 Windsurf 后，点击「+ 添加账号」输入名称保存
- **切换账号** — 点击卡片上的「切换」按钮
- **智能切换** — 点击「智能切换」自动选号（`free` 账号不参与）
- **删除账号** — 长按卡片 0.8 秒进入删除模式
- **导出/导入** — 账号列表标题右侧直接点击「↓ 导出」或「↑ 导入」
- **自动刷新倒计时** — 账号列表标题右侧显示下次自动刷新的剩余时间

## 功能效果

- **账号集中管理** — 多个 Windsurf 账号会显示在同一个列表中，方便查看和切换
- **用量一眼可见** — 每个账号显示日配额、周配额和剩余天数
- **自动更新用量** — 扩展会定时刷新账号用量，并显示下次刷新的倒计时
- **智能推荐账号** — 点击「智能切换」后，会自动选择更适合当前使用的账号
- **失效账号提示** — 如果账号登录状态失效，会在列表中显示为失效状态
- **数据备份恢复** — 可以导出 JSON 备份，也可以从 JSON 导入恢复账号数据

## 注意

- 仅支持 Windsurf 网页版
- 如果 Windsurf API 变更，protobuf 解析可能需要更新
- 部分登录流程可能依赖 cookies/localStorage 之外的状态
